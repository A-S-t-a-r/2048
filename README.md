## 2048
1. 项目简介：这是一个使用面向对象设计实现功能分离的小小小游戏——2048。 
2. 主要功能：通过键盘移动并且合并数字方块2、4、8...1024，达到最终合成2048的目的!
3. 项目目录结构解释：
        2048/（项目根目录）
    ├── 头文件（.h）───┐
    │   ├── GameBase.h       # 抽象基类，定义游戏基础数据和接口
    │   ├── GridGame.h       # 继承GameBase，实现网格游戏基础功能
    │   ├── Move.h           # 移动功能类，处理方块移动逻辑
    │   ├── Draw.h           # 绘制功能类，负责游戏界面渲染
    │   ├── Update.h         # 更新功能类，管理游戏状态更新
    │   └── Control.h        # 控制功能类，处理用户输入接口
    │
    ├── 源文件（.cpp）───┐
    │   ├── main.cpp         # 程序入口，初始化窗口和游戏循环
    │   ├── Move.cpp         # 实现四个方向移动的具体算法
    │   ├── Draw.cpp         # 实现游戏图形绘制和显示
    │   ├── Update.cpp       # 实现游戏状态更新逻辑
    │   └── Game.cpp         # 主游戏类，整合所有功能模块
    │
    ├── 依赖关系：
    │   GameBase → GridGame → (Move, Draw, Update, Control) → Game
    │
    └── 设计模式：
        使用虚拟继承的多继承架构，实现游戏功能模块化分离
